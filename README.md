x11_hash (v1.4)
===========================

Python module for Dash's X11 hashing for Ubuntu 22.04 and Python 3.10

Install
-------

```
apt install gcc python3-dev python3-pip

cd /home
git clone https://github.com/dashdot/python-x11_hash.git
cd python-x11_hash

python3 setup.py install
```
